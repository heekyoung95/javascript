### 함수

#### 콜백 함수
함수도 하나의 자료형이므로 매개변수로 전달이 가능하다.   
이렇게 매개변수로 전달하는 함수를 콜백 함수라고 한다.   
<br>

#### 콜백 함수에 익명함수 사용하기
```javascript
function callThreeTimes(callback) {
  for(let i = 0; i < 3; i++) {
    callback(i)
  }
}

callThreeTimes(function(i) {
  console.log(`${i}번째 함수 호출`)
})
```
