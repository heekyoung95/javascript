### Number 자료형으로 변환   
Number() 함수를 사용한다.  
또한 Number() 함수를 사용하지 않아도   
숫자가 아닌 다른 자료에서 0을 빼거나, 1을 곱하거나 1로 나누면   
자바스트립트가 내부적으로 자료형을 변환해준다.
```javascript
Number("123")
> 123

typeof(Number("123"))
> "number"


typeof(Number("123" - 0))
> "number"
```   
<br>

**\* 다른 문자가 들어있는 경우 NaN(Not a Number)를 반환한다.**   
**NaN**: 숫자이지만, 숫자로 나타낼 수 없는 숫자를 뜻한다.   
자료형은 숫자가 맞다.
```javascript
Number("$123")
> NaN

typeof(Number("123"))
> "number"
```

### String 자료형으로 변환   
String() 함수를 사용한다.  
또한 String() 함수를 사용하지 않아도   
문자열 연결 연산자(+)를 이용하면   
자바스트립트가 내부적으로 자료형을 변환해준다.
```javascript
123 + ""
> "123"

typeof(String(123 + ""))
> "string"
```
